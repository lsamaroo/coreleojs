(static) open(id, widthopt, heightopt, modalopt)

Opens a dialog on desktop browser or a side panel on mobile browsers. Note: For dialogs it will initialize the dialog if it was never initialized before.
Parameters:
Name Type Attributes Default Description
id String the id of the dialog or panel
width number <optional>
 a width to display the dialog
height number <optional>
 a height to display the dialog
modal boolean <optional>
 true true if the dialog should be modal, false otherwise. Defaults to true for the first time the dialog is opened if not specified.
